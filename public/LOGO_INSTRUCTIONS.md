# Logo Setup Instructions

To add your WASC logo to the website:

1. Place your logo image file in the `public` folder
2. Name it one of the following:
   - `logo.png` (recommended)
   - `logo.jpg`
   - `logo.svg`

3. The logo will automatically be displayed in the navbar
4. If the logo file is not found, a placeholder with "W" will be shown

## Supported Formats
- PNG (recommended for best quality)
- JPG/JPEG
- SVG (scalable vector graphics)

## Recommended Size
- Minimum: 64x64 pixels
- Recommended: 128x128 pixels or larger
- The logo will be automatically resized to fit

## Current Logo Location
The navbar will look for the logo in: `public/logo.png` (or .jpg, .svg)
